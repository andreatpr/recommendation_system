# Local Business Discovery Engine
Travel-oriented recommendation system using the Yelp Academic Dataset.

data/  
&nbsp;&nbsp;raw/          ← original Yelp JSON (not included on repo, but its linked)  
&nbsp;&nbsp;interim/

&nbsp;&nbsp;Week3 assignmente report         
README.md  

## Dataset
Yelp Academic Dataset — https://www.yelp.com/dataset  
Requires an academic free sign up. Original files are not included on this repo due tu restrictions of license.

Download an unzip on data/raw/

Obligatory files:
- yelp_academic_dataset_business.json
- yelp_academic_dataset_review.json
- yelp_academic_dataset_user.json
- yelp_academic_dataset_checkin.json

## How to Reproduce

### 1. Install dependencies
pip install -r requirements.txt

### 2. Upload JSON files to google Drive account
Route: drive/MyDrive/Yelp JSON/yelp_dataset/

### 3. Run notebook
Open notebooks/pre.ipynb on Google Colab and run all cells in order. Notebook mounts Drive files automatically.

### 4. Outputs
data/processed/business_clean.parquet       (150,346 rows, 58 cols)
data/processed/business_categories.parquet  (~750K rows,   2 cols)
data/processed/review_interactions.parquet  (6,990,280 rows, 3 cols)
data/processed/user_clean.parquet           (~1,987,896 rows, 4 cols)
data/processed/checkin_clean.parquet        (131,930 rows,  2 cols)
